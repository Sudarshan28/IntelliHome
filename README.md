# IntelliHome 🏠

IntelliHome is a smart home automation web application built using the MERN stack.

## 🚀 Live Demo
Frontend: https://intellihome-frontend.onrender.com

Backend API:
https://intellihome-backend.onrender.com

## ✨ Features

- User Authentication
- Secure Login & Registration
- Smart Device Dashboard
- Responsive UI
- MongoDB Cloud Database
- Real-time Backend Integration
- Cloud Deployment using Render

## 🛠️ Tech Stack

### Frontend
- React.js
- Vite
- Tailwind CSS

### Backend
- Node.js
- Express.js

### Database
- MongoDB Atlas

### Deployment
- Render

## 📂 Project Structure

```bash
IntelliHome/
│
├── frontend/
├── backend/
```

## ⚙️ Installation

### Clone Repository

```bash
git clone https://github.com/Sudarshan28/IntelliHome.git
```

### Frontend Setup

```bash
cd frontend
npm install
npm run dev
```

### Backend Setup

```bash
cd backend
npm install
npm run dev
```

## 🌍 Deployment

- Frontend deployed on Render
- Backend deployed on Render
- Database hosted on MongoDB Atlas
